Git Basics

  • Verzeichnis für das Git-Repository erstellen und initialisieren
agross@FIRIEL /scratch
$ mkdir first-steps

agross@FIRIEL /scratch
$ cd first-steps

agross@FIRIEL /scratch/first-steps
$ git init
Initialized empty Git repository in /scratch/first-steps/.git/
  • Datei "README" erzeugen
agross@FIRIEL /scratch/first-steps @master
± echo "Die Readme" > README
  • mit git status prüfen wie Git den aktuellen Stand des Repositories sieht, hier gibt es eine unbekannte Datei die  nicht unter Versionskontrolle ist (untracked file) 
agross@FIRIEL /scratch/first-steps @master?
± git status
# On branch master
#
# Initial commit
#
# Untracked files:
#   (use "git add <file>..." to include in what will be committed)
#
#       README
nothing added to commit but untracked files present (use "git add" to track)
  • Datei in den nächsten (ersten) Commit übernehmen, == Datei zum Index hinzufügen, == Datei "stagen"
agross@FIRIEL /scratch/first-steps @master?
± git add README
  • Die Datei wird durch den folgenden Commit unter Source Control gestellt.
agross@FIRIEL /scratch/first-steps @master^
± git status
# On branch master
#
# Initial commit
#
# Changes to be committed:
#   (use "git rm --cached <file>..." to unstage)
#
#       new file:   README
#
  • Commit durchführen, mit -m wird die Commitnachricht angegeben. Wird der Parameter -m nicht angegeben öffnet sich der konfigurierte $EDITOR. Alternativ kann per git config --global core.editor ein alternativer Editor eingestellt werden.
agross@FIRIEL /scratch/first-steps @master^
± git commit -m "Erster Commit"
[master (root-commit) 1dda153] Erster Commit
 1 files changed, 1 insertions(+), 0 deletions(-)
 create mode 100644 README
  • git log zeigt in der Historie des Repositories den ersten Commit
agross@FIRIEL /scratch/first-steps @master
± git log
commit 1dda15369571359615251a755b525a0bc77b3811
Author: Alexander Groß <agross@therightstuff.de>
Date:   Sat Aug 6 09:08:25 2011 +0200

    Erster Commit
  • git log --oneline zeigt pro Historieneintrag eine Zeile
agross@FIRIEL /scratch/first-steps @master
± git log --oneline
1dda153 Erster Commit
